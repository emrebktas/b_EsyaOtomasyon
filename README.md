# Beyaz Eşya Otomasyon Sistemi


## Proje Özeti 
Sistemimizde beyaz eşyalara ait özellikler arayüz eşliğinde veritabanında saklanıp işlem yapılıyor.


## Geliştirme Ortamı
Proje C# Forms ve mySql kullanılarak geliştirildi.
